# Engineering-Project
My project for Bachelor of Engineering

In order to run project from with maven use:
  
  <sub> mvn javafx:run </sub>
  
JAR generation:

1. Run 
  <sub> mvn clean </sub>

2. Run
  <sub> mvn install </sub>
  
3. In target folder there will be jar file with -shaded ending. It can be run with java -jar
